Parks Planner

Description:

Parks Planner helps you create your outdoor itinerary in seconds. Access tens of thousands of trails and national parks via the National Parks API and Trails API.

My Role:

This was a group project in the Edx Full-Stack Bootcamp, requiring the integration of 2 or more APIs. I delegated tasks among the group, created the UI using bootstrap, and collaborated with my team to integrate the APIs and add a bookings list using local storage.

Technologies Used:

HTML5, CSS, JavaScript, Bootstrap, National Parks API, Trail API

How to Use:

1. Select a US State or territory from the dropdown menu.

2. Click 'Search' to see all US National parks, reserves, and monuments in your selected state or territory.

3. Click 'View' on a park of your choosing, and you'll be prompted with park info, and nearby trails.

4. Select 'Book' to save the park into an itinerary.

5. Select 'Click to view bookings' to see your itinerary.



Reading Roundtable

Description:

Reading Roundtable is a full-stack express application integrated with a books API, that has a fully functional user database and requires sign up/login to use.

My Role:

This was a group project in the Edx Bootcamp. I worked on the UI using bootstrap and handlebars, and collorated on the signin/login functionality.

Technologies Used:

HTML5, CSS, Bootstrap, Handlebars.js, JavaScript, Express.js

How to Use:

1. Select 'Sign Up' and enter a valid email and password to create your account.

2. Search for titles in the search bar, and click the search button or press enter.

3. To Log out, select 'Log Out' in the navbar

4. Use the credentials you signed up with to login again in the future.



Weather Buddy

Description:

Weather buddy is a useful weather application that uses a weather API to provide forecasts all over the world, and allows you to save searches.

My Role:

I created this application.

Technologies Used:

HTML5, CSS, JavaScript, Bootstrap, Weather API

How to Use:

1. Click into the Search bar and search for a location using city name, zip code, or latitude/longitude.

2. Click 'Search' to see a 10 day forecast of your desired location.

3. Your searches will be saved in the 'Recent Searches' section.



Coding Quiz

Description:

Put your coding skills to the test with this 90 timed coding quiz. This quiz covers HTML, CSS, and JavaScript and reveals high scores after the quiz ends.

My Role:

I created this application.

Technologies Used:

HTML5, CSS, JavaScript

How to Use:

1. Select 'Begin Quiz' and the quiz will begin.

2. Select your answer and then click the 'Next' button. You may change your answer as needed before hitting next.

3. Continue answering questions until the quiz is completed, or time runs out.

4. Enter your initials and select 'Submit' to save your high score.

5. If you'd like to try again, click 'Coding Quiz Challenge' in the navbar.
